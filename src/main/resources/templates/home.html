Tips

Which ports need to be open?
ORPort: This is the "ORPort" that your relay will use to communicate with other relays in the Tor network.
ListenAddrPort: This port is used when running a bridge relay. It's the port that clients will connect to.
WebTunnel Ports: When configuring webtunnel, you need to open port 80 to issue a certificate. After that, you can close port 80 and use the port you configured in the webtunnel configuration file.
Limits of number of tor services on one IP address
Why Limits: Limits are set based on what Tor allows on one IP address. It's also recommended to only run one type of relay at a time.
Guard Relays: The limit for guard relays is 8.
Obfs Bridge: The limit for obfs bridge is 2.
WebTunnel: The limit for WebTunnel is 1.
Snowflake Proxy: The limit for snowflake proxy is 1.
Onion Service: Onion service is without a number limit.
What are the different types of tor services?
Middle Relay: This is a type of relay that is used to help users connect to the Tor network. It's the second relay that a user connects to.
Guard Relay: This is a type of relay that is used to help users connect to the Tor network. It's the first relay that a user connects to.
Bridge: This is a relay that is not listed in the public directory. It's used to help users bypass censorship.
Onion Service: This is a service that is hosted on the Tor network. It's used to host websites and other services.
Which bridge should I run?
Obfs4: Use this when you have an open IP.
WebTunnel: Use this when you have an open IP and a domain.
Snowflake: Use this when you don't have open IP.
